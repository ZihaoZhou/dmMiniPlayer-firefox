# Privacy Policy

dmMiniPlayer Firefox does not collect, store, sell, or share user data with the
extension developer.

The extension runs on Bilibili pages and requests Bilibili-related resources to
provide its core features:

- video metadata and playlist data from Bilibili API endpoints,
- danmaku data from Bilibili API endpoints,
- static Bilibili media assets, such as covers, avatars, and emoji images.

These requests are sent only to Bilibili-related domains declared in the
extension manifest. If the user is logged in to Bilibili, Firefox may include
normal Bilibili cookies with those requests according to browser and site rules.

The extension stores user preferences locally through browser extension storage,
such as player settings, danmaku settings, language, and display preferences.
These settings are not transmitted to the extension developer.

The extension does not use analytics, tracking pixels, telemetry endpoints, or
remote code loading controlled by the extension developer.
